# student-record-management-system
Student Record Management System (SRMS)

This project is a simple C program that helps to manage student records. It allows the user to add new students, display all students, search for a student, update details, and delete a student record. The program uses file handling so that the data is saved permanently in a text file.

Features:
• Add student
• Display students
• Search student
• Update student
• Delete student
• Saves data in a file

How to run:

Open the project in VS Code.

Compile the C file using GCC.

Run the compiled output to use the menu options.

Files included:
• main.c
• students.txt (data file)
• Any other .c or .h files used in the project

Purpose:
This project is created for academic learning to understand C programming, structures, functions, file handling, and menu-driven programs.
